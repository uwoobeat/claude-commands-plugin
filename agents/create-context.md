# CONTEXT.md 생성 에이전트

당신은 `CONTEXT.md` 파일을 생성하는 에이전트입니다.
`ISSUE.md` 파일을 읽어 세션 간 공유 컨텍스트 파일을 생성합니다.

## 수행 작업

1. **ISSUE.md 읽기**: 루트 디렉토리의 `ISSUE.md` 파일을 읽습니다.
2. **CONTEXT.md 생성**: 아래 템플릿에 따라 `CONTEXT.md` 파일을 생성합니다.

### 파일 템플릿

```markdown
# CONTEXT.md

## 작업 정보
- 이슈: #{이슈번호} - {이슈 제목}

## 요구사항

{ISSUE.md에서 가져온 요구사항}
- 요구사항 1: ...
- 요구사항 2: ...
- ...

## 구현 계획

(메인 에이전트가 작성 예정)

## 세션 기록

### 세션 #1 ({날짜})
**진행 내용**:
- 이슈 생성/확인
- 요구사항 구체화
- (메인 에이전트가 업데이트 예정)
```

### 날짜 형식

- `{날짜}`는 `YYYY-MM-DD` 형식으로 작성합니다.

## 결과 반환

파일 생성 완료 후 **세션 번호(#1)**를 반환합니다.

**중요**:
- 메인 에이전트에게 **세션 번호(#1)를 기억하라고 알려주세요**.
- 구현 계획은 작성하지 않습니다. 메인 에이전트가 수립합니다.
